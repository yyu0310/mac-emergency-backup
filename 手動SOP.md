# 我要手動做的事 — 備份 SOP

> 備份前親手執行。Claude 負責跑備份，你只要照下面操作。
> 原則：通電時間越短越好，備份完立刻關機。

---

## ☑️ 開機前

- [ ] 把外接硬碟插上**進水原機**
- [ ] 機殼縫縫沒有明顯水珠在流
- [ ] 心裡有數：這次開機只做備份，做完馬上關機

---

## ☑️ 開機 → 啟動 Claude Code

1. [ ] 按電源鍵開機（螢幕有色斑沒關係，看得到字就好）
2. [ ] 打開「**終端機**」(Terminal) — 不要用 VS Code（太重、發熱）
3. [ ] 在終端機輸入，每行按 Enter：
   ```
   cd /Volumes/YOUR_DRIVE_NAME
   claude
   ```
4. [ ] 進入 Claude Code 後，打字下指令（或用手機遠端下）：
   ```
   讀 claude-backup-instructions.md，照著幫我備份
   ```

> **萬一 `claude` 要求重新登入、又嫌麻煩** → 不用理它，直接在終端機跑腳本即可（不需登入）：
> ```
> bash /Volumes/YOUR_DRIVE_NAME/backup.sh
> ```
> 一樣完整備份。Claude 只是方便的檢查/回報層，不是必要的。

---

## ☑️ 備份進行中

- [ ] Claude 跑指令時若跳出**權限確認** → 按「同意」（手機端也可按）
- [ ] 等它自己檢查硬碟 → 跑 backup.sh → 驗證
- [ ] 等到它明確回報：**「可以安全關機了」**

---

## ☑️ 完成 → 關機

- [ ] 看到「可以安全關機了」+ 正常的大小數字
- [ ] 備份位置：`/Volumes/YOUR_DRIVE_NAME/EmergencyBackup/mac-backup-時間戳/`
- [ ] **關機**（外接碟是實體寫入，不用等上傳）
- [ ] 繼續立著、電風扇吹、保持關機

---

## ⚠️ 緊急中止（出現任一狀況立刻做）

- 機身**發燙 / 有異味 / 冒煙** → **立刻關機、拔掉電源線**，不要再開，直接送修
- 開機後螢幕全黑、完全沒畫面 → 別硬撐，直接帶去 MacBook 維修店請他們接外接螢幕備份
- 終端機說找不到外接硬碟 → 輸入 `ls /Volumes` 確認硬碟有掛上、名字沒打錯

---

## 之後

- [ ] 確認外接硬碟裡備份檔案都在
- [ ] 送 MacBook 維修店換螢幕
